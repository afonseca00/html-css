TORNAR IFRAME MAIS SEGURO
sandbox="sandbox" -> bloqueia todo e qualquer acesso que outros sites que estejam dentro de iframe tentem fazer com por exemplo um cadastro
referrerpolicy="no-referrer" -> não vai guardar nenhuma informação do usuario
exemplo:
<iframe src="paginas-extra/pagina004.html" sandbox="sandbox" referrerpolicy="no-referrer">
        Seu navegador não é compatível com isso...
    </iframe>

Quando precisamos que funcione o formulário
sandbox="allow-same-origin allow-forms allow-scripts"
allow-same-origin -> aceita coisas da mesma origem
allow-forms ->	aceita formulários
allow-scripts -> aceita scripts
exemplo:
 <iframe src="paginas-extra/pagina004.html" sandbox="allow-same-origin allow-forms allow-scripts" referrerpolicy="no-referrer">
        Seu navegador não é compatível com isso...
    </iframe>



FORMULÁRIOS
- para criar um formulario basico temos que começar com <form> e acabar </form>
- abrimos um <p> e metemos o nome e para ter a box para escrevermos metemos input:text
- para termos o botão para enviar metemos input:submit
- se o autocomplete estiver a chatear podemos acrescentar ao <from> o autocomplete="off" e fica assim <form autocomplete="off">
- para enviar os dados temos que acrescenatar o action="cadastro.php" ao form
EXEMPLO:
<form action="cadastro.php" autocomplete="off">
        <p>Nome: 
            <input type="text" name="nome" id="nome"></p>
        <p>Apelido: 
            <input type="text" name="apelido" id="apelido"></p>
        <p><input type="submit" value="Enviar"></p>
    </form>

-para relacionar o Nome a sua caixa precisamos de envelopara o Nome: para <label for="inome">Nome:</label> e meter lhe o nome do id do input (caixa)
- existem dois tipos de metodos get(padrão) e post 
- get aparece o que escrevemos na url
- post não aparece
- usar get quando dados não forem senciveis (ate 3000bits)
- usar post em campos sensiveis (se passar de 3000bits) e ficheiros
EXEMPLO:
<form method="post" action="cadastro.php" autocomplete="off">
        <p><label for="inome">Nome:</label> 
            <input type="text" name="nome" id="inome"></p>
        <p><label for="iapelido">Apelido:</label> 
            <input type="text" name="apelido" id="iapelido"></p>
        <p><input type="submit" value="Enviar"></p>
</form>

- para password temos que meter input:password
- para limpar temos que meter input:reset
- para não deixar avançar sem preencher o campo temos que meter required  
- para ter um minimo de letras  minlength=""
- para ter um maximo de letra maxlength=""
- se quisermos mostrar so não sei quantas letras metemos size=""
- se quisermos que na box esteja escrito alguma coisa antes de escrevermos placeholder=""
- podemos meter autocomplete nas labels para defenir o que são ver no exemplo
EXEMPLO:
<form method="post" action="cadastro.php" method="get" autocomplete="on">
        <p>
            <label for="iuti">Utilizador: </label>
            <input type="text" name="uti" id="iuti" required minlength="5" maxlength="15" 		size="10" placeholder="Nome" autocomplete="username">
        </p>
        <p>
            <label for="ipass">Password: </label>
            <input type="password" name="pass" id="ipass" required minlength="8" 			maxlength="20" placeholder="Min 8 letras" autocomplete="current-password">
        </p>
        <input type="submit" value="Enviar">
        <input type="reset" value="Limpar">
</form>


Elementos number, month, date e time
EXEMPLO:
<form action="cadastro.php" method="get" autocomplete="on">
        <p>
            <label for="inome">Nome: </label>
