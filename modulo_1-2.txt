NOTAS:
Vamos falar sobre box
heigth - altura
width - largura
border - borda (dentro do elemento)
padding - é o preenchimento entre o conteudo e o border
margin - é o espaço por fora do border
outline - linha por fora do border (fora do elemento)

TIPOS de BOX
box-level: 
(<div></div>) e todos os que precisao de quebra de linha
-> sempre se inicia numa linha nova
-> ocupa a width inteira
-> e para continuar o conteudo salta uma linha
inline-level: 
(<span></span>) e todos os que não precisao de quebra de linha
-> a box inicia se na mesma linha do conteudo
-> não ocupa a width toda
-> conteudo continua mal a box acaba
Exemplo:
h1 {
            background-color: lightgray;
            height: 300px;
            width: 300px;

            /*border-width: 10px;
            border-style: solid;
            border-color: darkslategray;*/

            border: 10px solid darkslategray;

            /*padding-top: 10px;
            padding-right: 10px;
            padding-bottom: 10px;
            padding-left: 10px;*/
	    
            padding: 10px 10px 10px 10px;

            /*margin-top: 20px;
            margin-right: 20px;
            margin-bottom: 40px;
            margin-left: 20px;*/

            margin: 20px 20px 40px 20px;

            /*outline-width: 5px;
            outline-style: dashed;
            outline-color: salmon;*/
            
            outline: 5px dashed salmon;
        }
padding: 10px (todos os varores ficam iguais)
padding: 10px 20px (o primeiro valor é para cima e baixo o segundo é direita esquerda)
padding: 20px auto 40px auto (centraliza a direita e a esquerda)
margin: auto; (para alinhar ao centro)
